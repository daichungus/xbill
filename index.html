XBill fixes

By Daichungus

Package Fixes

Wayland

Install gdk-pixbuf2 and gdk-pixbuf2-modules-extra through your package manager.

This way, xbill can load its xpm assets on Wayland.

KDE or DE's not based on GNOME

Optional, as it will reduce the obscene amount of errors in the terminal.

Otherwise it will cause some inconsistencies in the UI (especially for the buttons).

Install gnome-themes-extra.x86_64 through your package manager.

Building Fixes

This only applies to this version of xbill.

After autoconf and ./configure --enable-gtk, run

sed -i '2c\CFLAGS= -O2 -Wno-implicit-function-declaration' Makefile
